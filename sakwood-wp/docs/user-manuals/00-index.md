## Getting Started

If you are new to Sakwood, we recommend reading the manuals in this order:

1. Start with [Blog Post Management](./01-blog-management.md) to learn the basics
2. Read [Product Management](./02-product-management.md) for e-commerce features
3. Explore marketing tools with [Hero Slider](./03-hero-slider-management.md) and [Popup Management](./04-popup-management.md)
4. Learn customer management with [Wholesale](./05-wholesale-management.md) and [Dealer](./06-dealer-management.md) guides

## WordPress Admin Access

All features described in these manuals are accessed through the WordPress Admin Panel:

- **Development:** http://localhost:8006/wp-admin
- **Production:** [Your production URL]/wp-admin

## Language Support

The Sakwood platform supports two languages:
- **Thai (TH)** - Default language
- **English (EN)** - Secondary language

Most content types (products, blog posts) allow you to enter content in both languages.

## Need Help?

If you encounter issues not covered in these manuals:

2. Contact your technical administrator

## Quick Reference

### Common Admin Paths

| Task | Path |
|------|------|
| Products | WooCommerce > Products |
| Blog Posts | Posts > All Posts |
| Hero Slides | Hero Slides > All Slides |
| Popup Settings | Settings > Popup |
| Chat Settings | Settings > Chat Settings |
| Wholesale | Wholesale > All Applications |
| Dealer Applications | Dealer > Applications |

### Status Indicators

| Status | Color | Meaning |
|--------|-------|---------|
| Pending | Yellow | Awaiting review/action |
| Approved | Green | Approved but not yet active |
| Active | Green | Fully active |
| Rejected | Red | Declined |
