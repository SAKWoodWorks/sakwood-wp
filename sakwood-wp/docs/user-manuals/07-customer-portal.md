## Overview

The Customer Portal is a self-service area where registered customers can manage their account, view orders, manage addresses, and apply for wholesale/dealer status.

### Key Features

| Feature | Description |
|---------|-------------|
| **Account Dashboard** | Overview of customer activity |
| **Order History** | View all past and current orders |
| **Address Book** | Manage multiple delivery addresses |
| **Profile Settings** | Update personal information |
| **Wholesale Application** | Apply for wholesale pricing |
| **Dealer Application** | Apply for dealer status |

### Customer Access

Customers access the portal by:
1. Logging into their account on the website
2. Clicking **Account** in the header/menu
3. Or visiting `/[lang]/account` directly

## Accessing the Portal

### Customer Login

**From the Website:**

1. Click **Login** in the header
2. Enter username or email
3. Enter password
4. Click **Login**

**Login Options:**
- Username OR email accepted
- "Forgot Password" link available
- New customers can register during checkout

### Admin Access to Customer Accounts

Administrators can view customer accounts:

1. Go to **Users** > **All Users**
2. Click on customer name
3. View customer details and order history

**Note:** For security, admins cannot directly login as customers but can view their data.

## Account Dashboard

### Overview Tab

The main dashboard shows:

| Section | Description |
|---------|-------------|
| **Welcome Message** | Personalized greeting with customer name |
| **Account Type** | Retail, Wholesale, or Dealer status |
| **Quick Stats** | Total orders, total spent |
| **Recent Orders** | Last 3 orders with status |
| **Quick Actions** | Links to common tasks |

### Account Status Display

Customers see their account status:

| Status | Display | Benefits |
|--------|---------|----------|
| **Retail** | Standard customer | Regular pricing |
| **Wholesale** | Approved wholesale | 15% discount, Net-30 |
| **Dealer Silver** | Silver dealer | 20% discount |
| **Dealer Gold** | Gold dealer | 25% discount |
| **Dealer Platinum** | Platinum dealer | 30% discount |

## Order Management

### Viewing Orders

**From the Account Dashboard:**

1. Click **Orders** tab
2. View all customer orders in reverse chronological order
3. Each order shows:
   - Order number
   - Date
   - Status
   - Total amount
   - View Details button

### Order Status Meanings

| Status | Description | Customer Action |
|--------|-------------|-----------------|
| **Pending** | Order received, not yet processed | Wait for confirmation |
| **Processing** | Order being prepared | Delivery estimate coming |
| **On Hold** | Temporary pause | Contact support |
| **Completed** | Order delivered | No action needed |
| **Cancelled** | Order cancelled | Refund processed if applicable |
| **Refunded** | Refund issued | Check refund method |

### Order Details

Clicking **View Details** on an order shows:

**Order Information:**
- Order number and date
- Billing and shipping addresses
- Payment method
- Order status

**Items Ordered:**
- Product names and images
- Quantity
- Price per item
- Subtotal per item

**Order Totals:**
- Subtotal
- Shipping cost
- Discount applied (if any)
- Tax (if applicable)
- Total paid

**Tracking Information:**
- Tracking number (when shipped)
- Delivery link
- Estimated delivery date

### Reordering

From order details, customers can:

1. Click **Reorder** button
2. Items added to cart
3. Review and checkout

**Note:** Prices may have changed since original order.

## Address Management

### Address Book Overview

The address book allows customers to:

- Save multiple delivery addresses
- Set a default address
- Edit or delete addresses
- Select addresses during checkout

### Viewing Addresses

**From the Account Dashboard:**

1. Click **Addresses** tab
2. View all saved addresses
3. Each address shows:
   - Address type (Home, Office, Other)
   - Name and phone
   - Full address
   - Default indicator

### Adding a New Address

1. Click **Add New Address** button
2. Fill in required fields:

| Field | Required | Description |
|-------|----------|-------------|
| **First Name** | Yes | Recipient's first name |
| **Last Name** | Yes | Recipient's last name |
| **Company** | No | Business name (optional) |
| **Address Line 1** | Yes | Street address |
| **Address Line 2** | No | Apartment, suite, etc. |
| **City** | Yes | City or district |
| **State/Province** | Yes | Select from dropdown |
| **Postal Code** | Yes | ZIP/postal code |
| **Country** | Yes | Default: Thailand |
| **Phone** | Yes | Contact number for delivery |

3. Check **Set as default address** if preferred
4. Click **Save Address**

### Editing an Address

1. Find address in list
2. Click **Edit** button
3. Make changes
4. Click **Update Address**

### Deleting an Address

1. Find address in list
2. Click **Delete** button
3. Confirm deletion

**Note:** You cannot delete the default address. Set another as default first.

### Setting Default Address

1. Find address to make default
2. Click **Set as Default**
3. Address now used for:
   - New orders
   - Shipping calculations
   - Delivery estimates

## Wholesale Application

### Who Can Apply

Customers eligible for wholesale:

| Requirement | Details |
|-------------|---------|
| **Account Status** | Registered customer (anyone can apply) |
| **Business** | Must have valid business (tax ID requested) |
| **Location** | Must be in service area (Thailand) |
| **Approval** | Subject to review by admin |

### Application Process

**Step 1: Start Application**

1. From Account Dashboard, find **Wholesale Application** section
2. Click **Apply for Wholesale**
3. Application form opens

**Step 2: Complete Form**

| Field | Description |
|-------|-------------|
| **Company Name** | Registered business name |
| **Tax ID** | 13-digit Thai tax ID |
| **Business Address** | Registered business address |
| **Business Type** | Construction, retail, manufacturing, etc. |
| **Monthly Volume** | Estimated monthly purchase amount |
| **Additional Notes** | Any relevant information |

**Step 3: Submit**

1. Review all information
2. Click **Submit Application**
3. Confirmation message appears
4. Application status: **Pending Review**

### Application Status

After submitting, customers see:

| Status | Meaning | Next Steps |
|--------|---------|------------|
| **Pending** | Under review | Wait 2-3 business days |
| **Approved** | Application accepted | Wholesale pricing activated |
| **Rejected** | Application declined | See reason, can reapply after 90 days |

### Wholesale Benefits Once Approved

- 15% discount on all products
- Net-30 payment terms
- Credit limit (typically 50,000 THB)
- Priority customer support

## Dealer Application

### Prerequisites

Before applying for dealer status:

| Requirement | Value |
|-------------|-------|
| **Wholesale Status** | Must be active wholesale customer |
| **History** | 6+ months of wholesale orders |
| **Order Count** | Minimum 5 wholesale orders |
| **Order Total** | Minimum spend varies by tier |

### Dealer Tiers

Customers can apply for:

| Tier | Discount | Min Order | Requirements |
|------|----------|-----------|--------------|
| **Silver** | 20% | 50,000 THB | Active wholesale + 2 references |
| **Gold** | 25% | 100,000 THB | 6 months + 5 references + 100k capacity |
| **Platinum** | 30% | 200,000 THB | 12 months + 10 references + 500k capacity |

### Application Process

**Step 1: Check Eligibility**

1. From Account Dashboard
2. Look for **Dealer Application** section
3. If eligible, **Apply Now** button is active

**Step 2: Complete Application**

| Field | Description |
|-------|-------------|
| **Requested Tier** | Select Silver, Gold, or Platinum |
| **Storage Facility** | Describe your storage (20+ characters) |
| **Delivery Vehicles** | Describe transportation (20+ characters) |
| **Sales Capacity** | Monthly sales estimate (e.g., "100k", "500k") |
| **Dealer Experience** | Years in industry (30+ characters) |
| **Requested Territories** | Select provinces from list |
| **Trade References** | (Gold/Platinum) 2-5 business references |
| **Business References** | (Platinum) 3 business references with values |

**Step 3: Submit**

1. Complete all required fields
2. Click **Submit Application**
3. Application ID generated (e.g., DLR-xxxxxxxx)
4. Admin review begins

### Application Tracking

Customers can check status:

1. Return to Account Dashboard
2. Find **My Dealer Application** section
3. View current status and notes

| Status | Meaning |
|--------|---------|
| **Pending** | Under review |
| **Approved** | Approved - dealer benefits active |
| **Rejected** | Not approved - see reason |

## Support Guide

### Common Customer Questions

**"Why can't I see wholesale pricing?"**

Possible causes:
- Application not yet approved
- Not logged in
- Cache issue

Solutions:
- Check application status in account
- Log out and log back in
- Clear browser cache

**"My order status isn't updating"**

Timeline:
- Processing: 1-2 business days
- Shipping: 1-5 business days depending on zone

Contact support if:
- Status unchanged for 5+ days
- Expected delivery passed

**"How do I change my delivery address?"**

Options:
- Before order ships: Contact support immediately
- After order ships: Cannot change - arrange with delivery team

**"I forgot my password"**

Recovery:
1. Click **Login** in header
2. Click **Forgot Password?**
3. Enter email address
4. Check email for reset link
5. Create new password

### Helping Customers with Portal Issues

When supporting customers:

1. **Verify their identity** - Ask for email or order number
2. **Clarify the issue** - Understand what they're trying to do
3. **Guide step-by-step** - Provide clear instructions
4. **Offer alternatives** - If portal isn't working, offer to help directly

## Troubleshooting

### Issue: Customer Cannot Log In

**Possible Causes:**
- Wrong username/password
- Browser cache issues
- Account locked

**Solutions:**
1. Verify username/email is correct
2. Use "Forgot Password" link
3. Clear browser cache and cookies
4. Try different browser
5. Check if account is locked (admin)

### Issue: Addresses Not Saving

**Possible Causes:**
- Required fields missing
- Browser validation issues
- Network connectivity

**Solutions:**
1. Verify all required fields filled
2. Check postal code is valid format
3. Try different browser
4. Check internet connection
5. Note: Thai addresses may have special characters

### Issue: Orders Not Showing

**Possible Causes:**
- Customer logged into wrong account
- Cache showing old data
- Recent order not yet processed

**Solutions:**
1. Verify logged into correct account
2. Clear browser cache
3. Allow 15 minutes for new orders to appear
4. Check order confirmation email for order number

### Issue: Application Not Submitting

**Possible Causes:**
- Required fields incomplete
- Validation errors
- Network issues

**Solutions:**
1. Check all required fields filled
2. Verify phone number format
3. Check tax ID is 13 digits (for wholesale)
4. Try shorter text in description fields
5. Attempt on different device/network
